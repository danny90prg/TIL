# Problem
strange advertising
https://www.hackerrank.com/challenges/strange-advertising/problem

# Solve
```c++
int viralAdvertising(int n) {
    int sum = 0;
    int days[51] = {0,5,};
    sum += (int)(days[1]/2);
    for(int i = 2; i<=n; ++i){
        days[i] = (int)(days[i-1]/2) * 3;
        sum += (int)(days[i]/2);
    }
    return sum;        
}
```
