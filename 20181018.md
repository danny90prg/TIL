# Problem
Staircase
https://www.hackerrank.com/challenges/staircase/problem

# Solve
```c++
// Complete the staircase function below.
void staircase(int n) {
    for(int i = 0; i<n; ++i){
        for(int j = 1; j<n-i; ++j)  cout << " ";
        for(int j = 0; j<=i; ++j)   cout << "#";
        cout << endl;
    }
}
```
